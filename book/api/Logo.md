`Logo` (component)
==================

Web Page Logo

```xml
<Logo src='logo.gif' />
```

Properties
----------

### `className`

Css class name.

type: `string`


### `onTouchTap`

Function to execute when clicked.

type: `func`


### `src`

Source of the image file

type: `string`


### `tooltip`

The tooltip to show.

type: `string`


### `tooltipPosition`

Position of the tooltip.

type: `enum ('bottom'|'bottom-right'|'bottom-left'|'right'|'left'|'top-right'|'top'|'top-left')`
defaultValue: `'bottom'`
